### 1. 發布 API 到 Maven 倉庫（現在要做）

- ✅ **宣告協議標準** - 搶佔 NDS 協議命名
- ✅ **讓開發者提前使用** - 其他插件開發者可以引用 API
- ✅ **建立生態系統** - 讓大家開始開發 NDS-native 插件

**發布內容：**
- `nds-api-2.0.0.jar` - 只有接口定義，沒有實現
- 包含：`NoieDigitalSystemAPI` 接口、模型類、文檔

**使用方式：**
```groovy
// 其他開發者的 build.gradle
dependencies {
    compileOnly 'noie.linmimeng:noiedigitalsystem-api:2.0.0'
}
```

**特點：**
- ❌ **不能直接放在伺服器 `plugins/` 資料夾**
- ❌ **沒有 `plugin.yml`，不是插件**
- ✅ **只能作為依賴被其他插件引用**

**目的：**

**發布內容：**

**特點：**
